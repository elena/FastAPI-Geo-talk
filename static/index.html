"threading is for working in parallel,

async and await
					async def get_data():
    return {"item": "Bar"}

async def logic():
    data = await get_data()
    return {"Foo": data["item"]}

@app.get("/")
async def root():
    return await logic()
Normal functions are fine. Don't forget await, even when it's complex. Most cases would say this is overkill but there are cases, where we like performance. Let me share one.
